## Functions
### register

Register the caller as a game. Intended to be called by the
game contract itself from its constructor, so registration
authority cannot be spoofed by an EOA.


```solidity
function register() external;
```

### deregister

Deregister a game. Only the original registrar (tx.origin at
register time) may deregister.


