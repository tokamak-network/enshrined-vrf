# op-deployer

OP Deployer automates deploying new OP Stack chains. Documentation is maintained in the [ethereum-optimism/docs](https://github.com/ethereum-optimism/docs) repository ([docs.optimism.io](https://docs.optimism.io)). See [Chain Operators > Tools > Deployer](https://docs.optimism.io/operators/chain-operators/tools/op-deployer) for usage instructions.
